# Código de Luta

Site estático para GitHub Pages com programa de treino solo, biblioteca de vídeos, manual em PDF, painel de evolução e página dedicada de combos.

## Publicação

O projeto não depende de build. Publique a raiz do repositório no GitHub Pages usando a branch `main` e a pasta `/`.

## Observações

- O progresso e o acesso ficam salvos apenas no navegador via `localStorage`.
- O modo visitante permite abrir o conteúdo em uma hospedagem pública sem cadastro obrigatório.
- Para login real em produção, use backend com banco de dados e autenticação segura.
